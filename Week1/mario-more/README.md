# Mario – Double Pyramid (mario-more)

This folder contains the solution to the **Mario – More** problem set from **CS50x: Introduction to Computer Science**.

The program prints **two pyramids side by side** using hashes (#), inspired by Nintendo’s Super Mario Bros.

---

## Problem Description

- Prompt the user for the **height of the pyramids**.  
- Height must be a **positive integer** between 1 and 8.  
- Construct **two pyramids** separated by **two spaces**:  
  - The **first pyramid is right-aligned**  
  - The **second pyramid is left-aligned**
