# Mario – Single Pyramid (mario-less)

This folder contains the solution to the **Mario – Less** problem set from **CS50x: Introduction to Computer Science**.

The program prints a **right-aligned pyramid** of hashes (#), inspired by the pyramids in Nintendo’s Super Mario Bros.

---

## Problem Description

- Prompt the user for the **height of the pyramid**.  
- Height must be a **positive integer**.  
- Construct a **right-aligned pyramid** where the number of rows equals the entered height.
